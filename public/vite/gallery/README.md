# Gallery Folder

This folder is used to store photos when **NOT** using Google Photos integration.

## How to Use

1. Upload your photos (JPG, JPEG, PNG, GIF, WebP) directly to this folder
2. Photos will be automatically displayed in the gallery
3. Latest photos appear first

## Instructions

- Simply copy/upload image files to this directory
- The gallery will automatically detect and display them
- No additional configuration needed
- Photos are sorted by modification date (newest first)

## Supported Formats

- .jpg / .jpeg
- .png
- .gif
- .webp

## Note

If Google Photos integration is enabled in the admin settings, photos in this folder will be used as a fallback only when Google Photos is disabled.
